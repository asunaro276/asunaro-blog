Requirements

Requirement: モバイルハンバーガーメニュー

ビューポート幅 768px 未満ではデスクトップナビを非表示にし、ハンバーガーボタンでトグル開閉する縦積みナビに切り替える。

Scenario: モバイルでヘッダーを表示する

  • WHEN ビューポート幅が 768px 未満でページを表示する
  • THEN デスクトップ用ナビ（<nav>）が非表示になる
  • THEN ハンバーガーボタン（☰）がヘッダー右端に表示される
  • THEN ヘッダーは2カラム（ロゴ／ハンバーガー）に変わる

Scenario: ハンバーガーボタンをタップする

  • WHEN ハンバーガーボタンをタップする
  • THEN 縦積みナビが border-top: 1px solid var(--rule) 付きでヘッダー直下に展開する
  • THEN ボタンアイコンが ☰ から ✕ に変わる
  • THEN 再タップすると縦積みナビが閉じてアイコンが ☰ に戻る

Scenario: モバイルナビのアクティブ強調

  • WHEN 縦積みナビを表示する
  • THEN 現在のページに対応する項目が --accent 色 + border-left: 3px solid var(--accent) で強調される

Scenario: ページ遷移後にメニューを閉じる

  • WHEN View Transitions でページ遷移が完了する（astro:page-load）
  • THEN 縦積みナビが自動的に閉じる
  • THEN ボタンアイコンが ☰ に戻る
