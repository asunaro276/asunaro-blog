ADDED Requirements

Requirement: プロフィールセクションの表示

サイドバー上部にプロフィールセクションを設ける。アバター・名前・自己紹介文・GitHub/X リンクで構成する。

Scenario: サイドバープロフィールが表示される

  • WHEN 記事一覧またはカテゴリページのサイドバーを表示する
  • THEN "About" ラベルが上部に表示される
  • THEN アバター（88×88px、円形）が表示される
  • THEN 名前が Noto Serif JP 20px で表示される
  • THEN 自己紹介文が --fg-2 色で表示される
  • THEN "GITHUB ↗" と "X ↗" のリンクが横並びで表示される
