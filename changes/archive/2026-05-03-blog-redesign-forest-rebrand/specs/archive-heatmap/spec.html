ADDED Requirements

Requirement: 年×月グリッドヒートマップの表示

アーカイブウィジェットは年単位のセクションに分かれ、各年に 12ヶ月分のグリッドセルを表示する。セルの色の濃さはその月の投稿数に比例する。

Scenario: アーカイブヒートマップが表示される

  • WHEN サイドバーのアーカイブセクションを表示する
  • THEN "Archive" ラベルと "by month" が上部に表示される
  • THEN 年が降順で並ぶ
  • THEN 各年の下に 12ヶ月分のグリッドセルが 6×2 で表示される
  • THEN 投稿のある月のセルは --accent の濃淡で着色される
  • THEN 投稿のない月のセルは --bg-2 の薄い色で表示される

Scenario: 月セルにカーソルを乗せる

  • WHEN 投稿のある月のセルにカーソルを乗せる
  • THEN ツールチップで "YYYY.MM · N 件" が表示される

Scenario: 月セルをクリックする

  • WHEN 投稿のある月のセルをクリックする
  • THEN その月のアーカイブページ（/yearmonth/[yearmonth]/1）に遷移する

Requirement: 凡例の表示

ヒートマップ下部に投稿数と色の濃さの対応を示す凡例を表示する。

Scenario: 凡例が表示される

  • WHEN アーカイブセクションを表示する
  • THEN "少" ～ "多" のラベルとともに 4段階のカラーチップが表示される
